# 指针

## 1. 简介

每一个变量都存放在内存中的某个位置，内存中的这些位置都有地址，存放这些地址的变量就是指针。

```go
num := 12
p := &num
```

上面实例中，`num` 是一个数字，使用 `&` 符号取出 `num` 的地址， 存放到 `p` 的指针变量中， `p` 是 `int` 型变量。

```go
fmt.Print(p) 
// 0xc000014088
```

上面实例中，打印了 `p` 中存放的地址，每次运行时随机分配内存，所以每次结果不一样。

```go
fmt.Print(*p) // 12
```

用 `*` 符号来获取地址指向的值。

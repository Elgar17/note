# GO 语言

GO 语言开源于 2009 年，为了解决分布式开发服务的问题，很适合作为后端语言，很多大型互联网公司正在用 GO 代替 Java。

## 基本结构

```go
// 程序必有的包
package main

// I/O 模块
import "fmt"

// 程序开始的函数
func main() {
    fmt.Println("Hello, World!") // 输出
}
```

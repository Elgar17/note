# 同时发送一部请求

```js
(function(){
    var count = 0;
    setTimeout(()=>{ count++; handle() },1000)
    setTimeout(()=>{ count++; handle() },2000)
    function handle(){ if(count ==2){console.log(666)}}
})()
```

# vscode 开发技巧

## 1. 概述

vscode 是很轻量的开发工具，熟练使用开发工具，可以提高开发效率，下面介绍下 vscode 中的一些常用开发技巧。

## 2. 空格与Tab

缩进程序是需要注意的点是，使用空格还是 Tab，这俩很像，很多人不是很了解。

在 vscode 中使用 `Ctrl` + `Shift` + `P` 打开命令行， 输入 indent using space 来，使用 Tab 缩进还是空格缩进。
